## ðŸ†˜ Troubleshooting

**Image not showing?**
1. Check filename exactly matches: `neuronal-eye.png`
2. Verify file is in `assets/` folder (not subdirectory)
3. Check browser console for 404 errors (F12)
4. Clear browser cache (Ctrl+Shift+Delete)
5. If still missing, CSS fallback eye will display

**Image looks stretched?**
1. Ensure image is square (520x520px)
2. If rectangular, it will scale to fit the circle
3. Use `object-fit: contain` (already applied)

**Image lost on animation?**
1. This shouldn't happen - it's in a fixed circle
2. Check if image has transparent areas being animated
3. Hover effect will show it's working correctly
