## HOW TO USE ON A DISCOVERY CALL

### BEFORE THE CALL (30 seconds)
1. Open discovery-tool.html in your browser
2. Keep it open on your laptop
3. Position window so prospect can see you're taking notes

### DURING THE CALL (25-30 min)
Follow the 5-step flow:

**STEP 1: Company Basics (2 min)**
- Enter: prospect name, company, role, industry, size
- Click "Continue"
- Form validates required fields

**STEP 2: Pain Points & Goals (8-10 min)**
- Describe current state in their words (freetext)
- Select pain category from dropdown
- Checkboxes auto-populate with relevant pain points
- Document root cause
- Sidebar suggests next questions

**STEP 3: Technology Audit (5 min)**
- List all their tools (type + Enter to add each)
- Tool tags appear below input
- Click X to remove if needed
- Document integration gaps (which tools don't connect)

**STEP 4: Future Vision & Metrics (5 min)**
- Enter their vision for success
- Input metrics:
  - Team size affected
  - Hours/week currently spent
  - Average deal value
  - Volume per week
  - Monthly revenue at risk
  - Severity slider (1-10)
- ROI calculator updates in real-time
- Click "Generate Recommendations"

**STEP 5: Review & Generate (2 min)**
- Add any final notes
- Click "Generate All Documents"
- Proposal, SOW, Contract, Email template created instantly

### AFTER THE CALL (Same day, < 2 hours)
1. Review generated proposal
2. Download 3 documents (TXT format)
3. Copy email template
4. Send with documents attached
5. Data auto-saved to CRM

## PRO TIPS

### Tip 1: Take Notes Visibly
- Laptop on stand so they see you typing
- Builds trust, shows attention
- Validates their concerns

### Tip 2: Use Sidebar Questions
- Don't memorize, read them aloud
- Natural conversation flow
- Ensures you dig deep

### Tip 3: Quantify Everything
- Push for specific numbers
- "Around 10" → "Is it 8? 12? 15?"
- Numbers close deals

### Tip 4: Reference Their Words
- Tool quotes them back
- Proposal uses their language
- Feels personal, not templated

### Tip 5: Update CRM After Every Call
- Even if no proposal sent
- Track: "Not ready", "Exploring", "Budget issues"
- Build history for follow-ups

### Tip 6: Review Dashboard Weekly
- See which pain categories convert
- Notice industry patterns
- Adjust pitch based on data

### Tip 7: Batch Proposals Friday
- Calls Mon-Wed
- Generate proposals Friday afternoon
- Send Friday evening
- Prospect reviews over weekend
- Follow up Monday

## TROUBLESHOOTING

**Q: Data not saving?**
A: Check browser localStorage is enabled. Try different browser.

**Q: Can't add tools in Step 3?**
A: Type tool name, press Enter (not Tab/Space). Tag appears below.

**Q: Email template not copying?**
A: Click "Copy to Clipboard" button. Or manually select and copy text.

**Q: Want to reset everything?**
A: Click "Reset" button in header. Clears all data, starts fresh.

**Q: Export not working?**
A: CSV downloads to Downloads folder. Import manually to Google Sheets.

**Q: How to modify documents?**
A: Download TXT files → Open in Word/Google Docs → Edit → Save.

**Q: Can I use on mobile?**
A: Tool is responsive, works on iPad. Not ideal for phone during calls. Use laptop.
